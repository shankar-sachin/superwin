SuperWin is a fast, self-contained WinUI 3 desktop multi-tool — a global clipboard quick-picker, live system diagnostics, a rich-text notepad, color picker, hashing, and more, behind one clean dashboard.

Everything in the box

📋

Clipboard

Process-wide history with a global Win+Shift+V picker — search, pinned items, keyboard nav, and auto-paste back into your app.

📊

Diagnostics + Mini-monitor

Full system inventory plus live CPU, RAM, GPU/VRAM and disk throughput. A DPI-aware, colour-coded always-on-top mini-monitor.

📝

Notepad Super

A Word-Lite rich-text editor: fonts, bold/italic/underline, text & highlight colours, lists, alignment, and RTF/TXT save.

🎨

Color Picker

Full colour wheel, hex/RGB with one-click copy, and a screen eyedropper.

🔊

Volume Customizer

Per-app and master volume in exact percent and decibels (WASAPI).

🔒

Hash & Checksum

MD5 / SHA-1 / SHA-256 of text or files, powered by Windows CNG.

🌐

Network Info

Adapters, IPv4/IPv6, MAC addresses, and a quick ping.

🔁

Unit Converter

Length, mass, temperature, data-size — plus number bases.

🔑

Password Generator

Configurable length and character classes with a live strength readout.

✏️

Text Tools

Case conversion, trimming, counting, and other quick transforms.

☕

Keep Awake

Stop sleep / screen-off, optionally for a fixed duration.

🧩

JSON Formatter

Pretty-print, minify, and validate JSON with clear parse-error messages.

🆔

GUID Generator

Random version-4 GUIDs, single or in bulk, with case/hyphen/brace options.

📈

Graphing Calculator

Desmos-style: multiple colour-coded curves, pan & zoom, labelled grid — plus a built-in CAS that differentiates (d/dx) and simplifies.

🛡️

Security & Privacy

Cryptographically-secure tokens, a password strength meter, and one-click clipboard wiping.

📌

Always On Top

A global hotkey that pins or unpins any focused window above everything else — PowerToys-style.

⚙️

Settings

Theme (light/dark/system), launch at sign-in, and a hotkey you set by pressing the keys.

Get started in seconds

1

Download SuperWin_v2.3.1.exe and run it. It's a per-user install — no admin prompt.

2

Optionally add a desktop shortcut and launch at sign-in in the wizard.

3

Open SuperWin — it starts maximized and lives in the tray. Updates arrive automatically.
